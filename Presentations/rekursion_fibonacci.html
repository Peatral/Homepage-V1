Moritz und Christof

Aufgabe

Verfasst eine iterative Funktion, die die Fakultät einer Zahl berechnet. In Python.

Erkennt ihr eine Regel?

0, 1, 1, 2, 3, 5, 8, 13

Es ist eine Fibonacci-Folge

f(n) = f(n-1) + f(n-2)
n0 = 0; n1 = 1;

Definition

Eine Folge/Funktion wird in ihrer Definition noch einmal aufgerufen.
